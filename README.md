# DIY RV PRO

Configurable RV automation and monitoring platform.

## Project Objective
DIY RV PRO is a universal DIY RV controller platform built around an ESP32-C3 architecture with a touchscreen/web interface, configurable automation engine, telemetry visualization, and modular hardware abstraction.

The platform is intended to support:
- Generator automation/control
- HVAC multi-zone control
- Shore/inverter/power management
- Battery and solar monitoring
- Tank monitoring
- User-programmable automation
- Historical telemetry and graphing
- Mobile/web interface
- Setup/configuration wizard

## Architecture
- Frontend: React/Vite
- Backend Bridge: Node/WebSocket
- Firmware: ESP32-C3
